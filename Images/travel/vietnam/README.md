## Notes
- Filenames are case-sensitive on GitHub Pages — keep the `.JPG` uppercase (the two `PXL_` files use lowercase `.jpg`, as named by the camera).
- HEIC files must be converted to JPG/PNG for browsers to display them.
- The page already lays out the photos in a responsive grid; no other change needed.
- Each photo also needs a matching `webp/<name>.webp` — drop the originals in this folder and the WebP versions can be generated from them (resized to 1600px wide, with EXIF rotation baked in).
